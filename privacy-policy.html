OTTDS - OTT Dual Subtitles

개인정보처리방침 / Privacy Policy

1. 개요

OTTDS(OTT Dual Subtitles)는 OTT 플랫폼(쿠팡플레이, 웨이브, 티빙, 디즈니+, 넷플릭스)에서 원문 자막과 번역 자막을 동시에 표시하는 Chrome 확장프로그램입니다.

OTTDS는 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.

2. 수집하는 데이터

OTTDS는 다음의 데이터만 처리하며, 모든 처리는 사용자의 브라우저 내에서만 이루어집니다:

데이터 유형 용도 저장 위치
자막 텍스트 번역 API로 전송하여 번역 결과 표시 브라우저 메모리 (세션 종료 시 삭제)
확장프로그램 설정 사용자 설정 유지 (언어, 폰트 등) Chrome Storage (로컬)
번역 캐시 동일 자막 재번역 방지 브라우저 메모리 (세션 종료 시 삭제)

3. 수집하지 않는 데이터

OTTDS는 다음 데이터를 일체 수집하지 않습니다:

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 로그인 정보 또는 인증 데이터
  • 브라우징 기록 또는 방문 URL
  • OTT 계정 정보 또는 시청 기록
  • 쿠키 또는 추적 데이터
  • 기기 정보 또는 위치 정보

4. 외부 서비스 통신

OTTDS는 자막 번역을 위해 다음 외부 서비스와 통신할 수 있습니다:

  • Google 번역 API — 자막 텍스트만 전송되며, 개인정보는 포함되지 않습니다.

전송되는 데이터는 자막 원문 텍스트뿐이며, 사용자를 식별할 수 있는 어떠한 정보도 포함되지 않습니다.

5. 권한 사용 목적

권한 사용 목적
host_permissions (OTT 사이트) 자막 데이터를 가로채서 번역하기 위해 필요합니다. 해당 OTT 사이트에서만 동작하며, 다른 사이트에는 접근하지 않습니다.
storage 사용자 설정(언어, 폰트 크기 등)을 로컬에 저장하기 위해 사용합니다.
activeTab 현재 활성 탭에서 자막 오버레이를 표시하기 위해 사용합니다.

6. 데이터 보안

모든 데이터 처리는 사용자의 로컬 브라우저 내에서 이루어지며, 외부 서버에 저장되지 않습니다. 번역 API와의 통신은 HTTPS를 통해 암호화됩니다.

7. 아동 보호

OTTDS는 13세 미만의 아동으로부터 개인정보를 수집하지 않습니다. 이 확장프로그램은 개인정보를 수집하지 않으므로 모든 연령의 사용자가 안전하게 사용할 수 있습니다.

8. 변경 사항 고지

본 개인정보처리방침이 변경될 경우, 확장프로그램 업데이트를 통해 고지합니다.

1. Overview

OTTDS (OTT Dual Subtitles) is a Chrome extension that displays original and translated subtitles simultaneously on OTT platforms (CoupangPlay, Wavve, Tving, Disney+, Netflix).

OTTDS does not collect, store, or transmit any personal information.

2. Data We Process

OTTDS only processes the following data, and all processing occurs within the user's browser:

Data Type Purpose Storage Location
Subtitle text Sent to translation API to display translated results Browser memory (cleared on session end)
Extension settings Persist user preferences (language, font, etc.) Chrome Storage (local)
Translation cache Prevent re-translating identical subtitles Browser memory (cleared on session end)

3. Data We Do Not Collect

OTTDS does not collect any of the following:

  • Personally identifiable information (name, email, phone, etc.)
  • Login credentials or authentication data
  • Browsing history or visited URLs
  • OTT account information or viewing history
  • Cookies or tracking data
  • Device information or location data

4. External Service Communication

OTTDS may communicate with the following external services for subtitle translation:

  • Google Translate API — Only subtitle text is transmitted. No personal information is included.

The only data transmitted is the raw subtitle text. No information that could identify the user is ever sent.

5. Permission Usage

Permission Purpose
host_permissions (OTT sites) Required to intercept subtitle data for translation. Only operates on specified OTT sites; does not access any other websites.
storage Used to save user settings (language, font size, etc.) locally.
activeTab Used to display the subtitle overlay on the currently active tab.

6. Data Security

All data processing occurs within the user's local browser and is never stored on external servers. Communication with the translation API is encrypted via HTTPS.

7. Children's Privacy

OTTDS does not collect personal information from children under 13. Since this extension does not collect personal information, it is safe for users of all ages.

8. Changes to This Policy

If this privacy policy is updated, changes will be communicated through extension updates.
